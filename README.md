# Heart-Attack-Risk-Prediction-Model-ML
An end-to-end Machine Learning project for Heart Disease Prediction using the UCI Heart Disease Dataset, including EDA, data preprocessing, feature engineering, Logistic Regression, model evaluation, and upcoming Decision Tree, Random Forest, and XGBoost implementations.
# ❤️ Heart Attack Risk Prediction Model - ML

An end-to-end Machine Learning project focused on predicting Heart Attack Risk using the UCI Heart Disease Dataset. This project demonstrates the complete ML workflow from data preprocessing and exploratory data analysis (EDA) to model training and evaluation.

---

## 📌 Project Overview

The objective of this project is to build a machine learning model that can predict the risk level of heart disease based on a patient's medical information.

This project follows industry-standard machine learning practices including:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Model Training
- Model Evaluation

---

## 📂 Dataset

- **Dataset:** UCI Heart Disease Dataset
- **Source:** UCI Machine Learning Repository

The dataset contains medical attributes such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Maximum Heart Rate
- Exercise-Induced Angina
- ST Depression (Oldpeak)
- Number of Major Vessels (CA)
- Thalassemia
- Target (Heart Disease Risk)
